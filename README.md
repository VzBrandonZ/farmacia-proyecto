# Farmacia-proyecto
Desarrollo web backend y frontend para inventario de farmacia con MERN Stack

## Secciones
- Medicinas
- Categorias
- Lotes
- Entradas 
- Salidas
- Administracion de usuarios
- Reporte
- Alertas
